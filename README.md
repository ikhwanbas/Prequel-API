# Installation
In order to start this project for the first time, you should run
```
npm install
```
To start this project, run
```
npm start
```
To start with nodemon, use
```
npm run dev
```

## Environment variables
This project uses environment variable as follows
```
MYSQL_PASSWORD=""
MYSQL_DATABASE=""
MYSQL_USER=""
MYSQL_HOST=""
PORT=""
JWT_SECRET=""
```
